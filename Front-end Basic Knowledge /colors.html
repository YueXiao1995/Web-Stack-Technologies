In HTMl, a color can also be specified using RGB values, HEX values, HSL values, RGBA values and HSLA values

#rrggbb: rr, gg, bb are hexadecimal values between 0 and 225

rgb(255, 99, 71)

rgb: each parameter (red, green, blue) defines the intensity of the color between 0 and 225

#ff6347

hsl (hue, saturation, lightness)

hsl(9, 100%, 64%)

rgba (red, green, blue, alpha)): is an extension of RGB color value, the alpha channel specifies the opacity for a color, it is between 0.0 and 1.0

rgba(255, 99, 71, 0.5)

hsla (hue, saturation, lightness, alpha)

hsla(9, 100%, 64%, 0.5)
